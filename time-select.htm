jQuery TimeSelect Plugin

Usage

    $('#textbox1').timeSelect();

    $('#textbox2').timeSelect({
        maxResults: 5,
        minsInterval: 10,
        error: function (element, selectedTime) {
            alert('You chose "' + selectedTime + '", which is an invalid option.')
        },
        success: function (element, selectedTime) {}
    });

Options

maxResults - (number) Maximum number of results to be shown in the dropdown list. Default is 10
minsInterval - (number) The interval between minutes. Default is 5 minutes.
error - (function) This callback function is fired when the user enters a value that is not present in the dropdown list. The callback is passed two parameters (element, selectedTime).
success - (function) This callback function is fired when the user either selects a value from the dropdown list or manually enters a value that was present in the dropdown list. The callback is passed two parameters (element, selectedTime). PS: Both the callbacks are fired only on "onchange" of the input field; not immediately when a value is selected or typed in.
